## `📚` <stdio.h>

# `⚙️` printf
  │
  ├── **`Affiche du texte formaté sur la sortie standard.`**
  ├── `Utilise des spécificateurs (%d, %s, etc.).`
  └── **Exemple d'utilisation**:

```c
    <stdio.h>

    int main()
    {
      char* string = "Hello World !";
      printf("Message: %s", string);
      
      return 0;
    }
```
